About

I'm Yash — a software engineer at HCLTech and a student at BITS Pilani, spending most of my actual free time trying to make free-tier cloud infrastructure do things it explicitly wasn't designed for.

My running joke of a specialty is the "GitHub Actions daemon" pattern: instead of paying for a VPS, I self-trigger workflows that run for ~5h50m, exit cleanly right before the runner's hard timeout, and kick off the next run — a genuinely free, always-on service with zero infrastructure to maintain. I've used it to build a small fleet of real-time Discord profile widgets synced to Spotify, Last.fm, AniList, Genshin Impact and live rocket launches.

Outside of that: cloud & Windows administration, Python automation, and enough scattered hobby projects — fractal generators, browser-native OS simulations, wearable data bridges — to make my GitHub contribution graph look more organized than my actual thought process.
